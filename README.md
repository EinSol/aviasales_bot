## Aviasales Bot

This Bot helps you organize your future travel

## How it works
# Start menu:
  - Choose destination country
  - Choose departure city
  - specify date of flight
# Additional options:
  - hotel class
  - Meals in the hotel
  - Number of children
  - Number of adults
    
# Results menu:
- Choose hotel 
- Fill out the application
- Agent will contact you
